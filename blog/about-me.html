Tentang Saya

Artikel ini adalah awal dari perjalanan saya menulis. Nantinya, saya akan banyak berbagi pemikiran dan refleksi seputar dunia keuangan, investasi, ekonomi, dan bagaimana membangun hidup yang lebih sadar secara finansial. Tapi sebelum itu, saya ingin kamu tahu siapa saya sebenarnya, dan dari mana saya memulai.

Nama saya Ade. Bukan miliarder. Bukan lulusan Ivy League. Bukan anak konglomerat. Saya hanya seseorang yang pernah salah jalan, tapi sekarang memilih jalan yang benar — walaupun pelan, sepi, dan kadang tidak terlihat.

Di masa lalu, saya gagal menyelesaikan dua universitas. Saya pernah bergabung dengan lingkungan yang salah — geng jalanan, narkoba, utang pinjol. Saya membuat kecewa banyak orang, terutama orang tua saya. Tapi semua itu menjadi titik balik yang mengajarkan saya satu hal penting:

Hidup bisa rusak. Tapi arah bisa dibenarkan — asal kita punya keberanian untuk sadar dan bertanggung jawab.

Dari Kegelapan Menuju Kesadaran

Saya tidak bangkit karena motivator. Saya bangkit karena saya sadar kalau saya terus begini, saya tidak akan pernah punya kendali atas hidup saya sendiri.

Dari situ saya mulai belajar soal keuangan. Bukan sekadar nabung, tapi membangun kekayaan secara sadar dan logis. Saya berhenti utang. Saya berhenti konsumsi barang demi gengsi. Saya mulai investasi. Bukan untuk kaya besok, tapi supaya saya tidak miskin selamanya.

Saya hidup di Bali — bukan kota murah. Gaji saya setara UMR. Tapi saya tetap sisihkan untuk masa depan. Sementara banyak orang kerja demi bayar cicilan, saya kerja demi punya pilihan.

Mindset yang Saya Bangun

Saya sadar bahwa kebanyakan orang hidup untuk "kelihatan berhasil", bukan benar-benar ingin berhasil. Mereka kejar rumah, nikah, mobil — tapi seringkali terjebak dalam utang yang tidak pernah selesai. Saya tidak ingin hidup seperti itu.

Saya ingin hidup yang tenang, bukan hidup yang terburu-buru.

Saya sering ditanya: “Kapan punya rumah?”, “Kapan nikah?”, “Kenapa nggak bantu keluarga lebih banyak?” Jawaban saya sederhana: Saya sedang menanam.

Saya sedang menanam sesuatu yang tidak tumbuh dalam seminggu, atau sebulan. Tapi saat ia tumbuh, ia takkan bisa ditebang oleh waktu atau krisis.

Filosofi Hidup Saya

  • Lebih baik lambat tapi dalam, daripada cepat tapi rapuh.
  • Uang bukan tujuan, tapi alat untuk merdeka.
  • Investasi terbesar adalah perubahan mindset.
  • Gagal itu proses. Tapi stuck itu pilihan.
  • Saya tidak hidup untuk memuaskan ekspektasi sosial. Saya hidup untuk membangun fondasi.

Apa yang Saya Lakukan Sekarang

Saya sedang membangun portofolio sebagai investor. Masih kecil, belum stabil, belum spektakuler — tapi punya arah.

Saya sedang belajar berbisnis. Masih di tahap awal, masih merangkak — tapi tidak menyerah.

Saya tidak punya banyak, tapi saya punya kontrol. Dan buat saya, itu adalah bentuk kekayaan paling mendasar.

Penutup

Banyak orang ingin cepat terlihat sukses. Saya tidak.

Saya ingin tumbuh secara utuh. Saya ingin menua tanpa panik soal uang. Saya ingin hidup yang tidak sekadar lulus, tapi bermakna.

Jika ada satu hal yang bisa kamu ambil dari cerita saya, itu ini:

Kesadaran adalah awal dari segala kemajuan. Hidup boleh dari nol, tapi jangan pernah berhenti bertumbuh.
