Proyek Utama

Main Project

BloomBoard — Platform Pelacak Portofolio & Lab Trading
BloomBoard adalah platform pelacak portofolio bergaya Bloomberg yang saya rancang untuk memungkinkan pengguna menyimpan dan mengelola data investasi secara mandiri melalui Supabase. Fitur utama meliputi: kustomisasi portofolio, modul "Lab" untuk backtesting dan eksperimen strategi dengan data historis, eksekusi order otomatis via script/robot, koneksi langsung ke wallet atau broker untuk trading nyata, serta agregator berita cerdas yang menandai informasi relevan berdasarkan komposisi portofolio pengguna. Sistem ini dirancang agar fleksibel — pengguna dapat menyesuaikan indikator, alert, dan tampilan sesuai kebutuhan investasi mereka.

BloomBoard — Portfolio Tracker & Trading Lab
BloomBoard is a Bloomberg-style portfolio tracker I designed to let users independently store and manage investment data via Supabase. Key features include customizable portfolios, a "Lab" module for backtesting and strategy experimentation using historical data, automated order execution via scripts/bots, direct wallet or broker connections for live trading, and an intelligent news aggregator that surfaces relevant coverage based on each user’s portfolio composition. The system is flexible—users can tailor indicators, alerts, and views to their investment needs.
